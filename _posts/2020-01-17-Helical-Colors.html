Helical conduit, colored in.

As a brief addendum to my last post: It is now possible to create helical revolutions using sketch within SpaceClaim. As with extrusions, the different faces can be reliably identified and turned into Named Selections - or, as in the example shown (an interlocking conduit), the faces can be colored individually, which I think makes the geometry a little easier to interpret. (Although, perhaps the purple-blue scale isn't the best.)

The next step is to find a way to split or segment the helix so that sections can be targetted for analysis later on in ANSYS Static Structural. A helix is basically just a spring, but the interlocking makes things interesting...
